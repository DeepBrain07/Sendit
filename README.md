# 🚀 SendIt.

**Peer-to-Peer Delivery Platform — Interswitch Hackathon (March 2026)**

---

## 📌 Overview

SendIt is a peer-to-peer delivery platform that connects people who need to send packages with verified travelers already heading in the same direction.

It enables affordable, secure, and trustworthy delivery by leveraging everyday travel.

---

## 💡 The Problem

- Informal delivery systems lack trust and accountability  
- No protection when deliveries fail  
- Pricing is inconsistent and sometimes exploitative  
- Courier services are expensive and rigid  

Users are forced to choose between:

> ❌ Cheap but risky  
> ❌ Safe but expensive  

---

## 🎯 The Solution

- 🔐 **Escrow payments** for security  
- 👤 **User verification** (ID + selfie)  
- 💬 **Real-time chat** for coordination  
- 💸 **Flexible pricing** (accept or bid)  
- ⭐ **Ratings system** for trust  

---

## ⚙️ How It Works

### 📦 For Senders

1. Post a delivery request  
2. Receive bids or acceptances  
3. **Select a carrier**  
4. Fund escrow  
5. Track and confirm delivery  
6. Release payment  

### 🚚 For Carriers

1. Browse delivery requests  
2. Accept price or submit a bid  
3. Get selected by sender  
4. Pickup and deliver package  
5. Receive payment

## 🛠️ Tech Stack

### Frontend
- **Framework:** React (with TypeScript)  
- **Styling:** Tailwind CSS  
- **Icons:** Iconify  
- **Build Tool:** Vite  

### Backend
- **Language:** Python  
- **Framework:** Django REST Framework (DRF)  
- **Database:** PostgreSQL  
- **API Testing:** Postman  

### Services
- **Payments:** Interswitch API  
- **Media Storage:** Cloudinary  
- **Real-time:** Socket.io  
- **Notifications:** Firebase

## 🧠 Key Product Decisions

- Two-stage verification  
- Escrow system for trust  
- Auto-release after 24 hours  
- Marketplace pricing (accept or bid)
